# Banco de Dados

São sistemas organizados, usados para coletar, armazenar e gerenciar dados.

**Os principais componentes de um banco de dados são:**

* **Dados**: São as informações armazenadas no banco de dados. Podem assumir várias formas, como números, texto, imagens, entre outros.

* **Tabelas**: São estruturas retangulares compostas por linhas e colunas usadas para armazenar os dados. Cada linha em uma tabela representa uma entrada de dados específica, enquanto cada coluna contém um tipo particular de dado.

* **Colunas**: Cada coluna em uma tablema é chamada de campo e representa uma categoria específica de dados.

* **Linhas**: Cada linha em uma tabela é chamada de registro e representa uma instância específica de dados, contendo informações sobre um item específico.

* **SGBD**: Significa "Sistema Gerenciador de Banco de Dados", e é um conjunto de softwares responsáveis pelo gerenciamento de um banco de dados.

## Banco de Dados Relacionais

São sistemas de gerenciamento de banco de dados (SGBDs) que organiza dados em tabelas.
